this folder contains jpg images of 2048*1024 pixels.
define xxxx as the name of the celestial object (ex: earth).
xxxx_normal_map.jpg which contain the normal map.
xxxx_lights_map.png is a mask which contain only the lights as view during night time with transparency for dark area.
xxxx_map.jpg which contain the day colored map.
xxxx_specular_map.jpg which contain the specular map, which is mainly intended to define reflection over water area.
